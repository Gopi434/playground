# **App Name**: Gmail Sign-Up MVP

## Core Features:

- Email Input: A single input field that accepts only Gmail addresses.
- Client-Side Validation: JavaScript validation to ensure the email ends with '@gmail.com'.
- Submit Button: A button to submit the email address.
- Webhook Integration: On submit, the email address is sent as a POST request to the specified webhook URL (WEBHOOK_URL_HERE).
- Success Message: A message displayed to the user upon successful submission.

## Style Guidelines:

- Primary color: Light, desaturated blue (#A7D1EA) to evoke trust and reliability.
- Background color: Very light blue (#F0F8FF), visibly of the same hue as the primary color.
- Accent color: Slightly darker blue (#75A9D1), for contrast on interactive elements.
- Font: 'Inter' sans-serif font for clear and modern readability.
- Centered form with basic styling for a clean and focused user experience.
- Subtle transition animations on button hover and form submission for user feedback.
